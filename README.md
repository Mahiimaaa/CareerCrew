# CareerCrew 
### AI-Powered Job Application Assistant | Agentathon 2026

Paste any job description and instantly get everything you need to apply with confidence.

##  Features
-  Personalized Cover Letter
-  Skills Checklist extracted from the job description
-  Interview Tips & Strategy

##  Tech Stack
- **Frontend:** HTML, Tailwind CSS, JavaScript
- **Automation:** n8n Cloud (4-node workflow)
- **AI Model:** OpenAI GPT-4o-mini
- **Fonts:** Manrope + Inter

##  How to Run
1. Open `careercrew_newui.html` via VS Code Live Server
2. Paste any job description
3. Click Generate — done!
